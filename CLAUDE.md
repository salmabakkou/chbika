# CHBIKA - Project Context 

You are working as a senior Front-End developer and UI/UX designer for the CHBIKA brand.

Before generating any code, understand the brand identity and the user's vision.

The user is a Moroccan creator building a luxury crochet brand inspired by Italy and Mediterranean culture.

Your goals are:

1. Create a unique luxury experience.
2. Maintain a strong visual identity.
3. Generate production-quality code.
4. Keep the code simple and understandable.

### UI Inspiration

The website should feel like a luxury fashion brand.

Keywords:

* Dolce Vita
* Sicily summer
* Mediterranean sea
* Luxury resort
* Italian villa
* Sunshine
* Flowers
* Crochet details
* Handcrafted products

### Homepage Experience

The first impression must be emotional.

Hero section:

* Full screen image
* Elegant female model
* Mediterranean landscape
* Natural sunlight
* Vibrant summer colors
* Minimal text

Sections:

* Hero
* New Summer Collection
* Best Sellers
* Brand Story
* Editorial Gallery
* Newsletter
* Footer

### E-commerce Features

Admin:

* Add products
* Edit products
* Delete products
* Manage orders

Customer:

* Browse collections
* Product details
* Add to cart
* Wishlist
* Checkout

### Code Quality

Every component must:

* Be responsive
* Have clean naming
* Be reusable
* Follow modern React practices

Avoid generic e-commerce designs.

CHBIKA must look like a premium Mediterranean fashion house.
