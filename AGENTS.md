# CHBIKA - AI Development Guide

## Project Overview

CHBIKA is a luxury crochet e-commerce website inspired by the Mediterranean summer, Sicily, Italian coastal lifestyle, and Moroccan craftsmanship.

The brand identity combines:

* Sicilian Dolce Vita
* Mediterranean blue sea
* Vibrant flowers
* Lemon gardens
* Luxury fashion editorial aesthetics
* Handmade Moroccan crochet

The website must feel:

* Premium
* Elegant
* Feminine
* Warm
* Colorful
* Minimal and modern

## Tech Stack

* Next.js 15 (App Router)
* JavaScript (No TypeScript)
* Tailwind CSS
* Supabase (Database + Storage)
* Redux Toolkit for cart and wishlist
* Framer Motion for animations

## Development Principles

Always:

* Write clean and maintainable code
* Create reusable React components
* Follow responsive mobile-first design
* Use semantic HTML
* Optimize images and performance
* Use meaningful component names

Never:

* Use TypeScript
* Write large components with multiple responsibilities
* Use unnecessary libraries
* Create duplicate code

## Folder Architecture

Use the existing project structure:

* app for routes
* components for reusable UI
* services for Supabase requests
* store for global state
* hooks for custom hooks
* utils for helper functions

## Design Philosophy

The design must communicate a luxury summer experience.

Inspired by:

* Sicily
* Taormina
* Mediterranean coast
* Italian summer fashion editorials

Colors:

* Mediterranean Blue
* Turquoise Sea
* Lemon Yellow
* Bougainvillea Pink
* Olive Green
* Warm White

UI Style:

* Large images
* Lots of whitespace
* Elegant typography
* Smooth animations
* Rounded soft elements
* Premium e-commerce feeling

## Coding Standards

Use:

* Functional components
* React hooks
* Async/await
* Modular CSS using Tailwind utilities

Prefer simple solutions before complex ones.
Keep components scalable.
